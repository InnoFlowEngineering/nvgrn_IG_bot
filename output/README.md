# Output Folder

This directory contains subfolders for organizing Instagram content.

## Structure

```
output/
├── scheduled/   # Posts scheduled for future publication
├── posted/      # Published content (archives)
└── drafts/      # Draft posts not yet scheduled
```

## Purpose

In a production system, these folders would store:
- Media files (images, videos)
- Metadata JSON files
- Publication logs
- Analytics data

## Demo Note

In this demo version, these folders are placeholders. In production:
- Files would be stored here before upload to Instagram
- Metadata would be saved as JSON
- Published content would be archived
- Analytics would be tracked per post
